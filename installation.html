Installation¶

alignparse requires Python 3.6 or higher.

The easiest way to install alignparse is from PyPI using pip with:

pip install alignparse

The source code for alignparse is available on GitHub at https://github.com/jbloomlab/alignparse.

In order to use the alignparse.minimap2 module, you need to install the minimap2 executable. You can do that via the bioconda recipe or from the program’s release page as described here.
